# 🌤 Weather Dashboard

## 📌 Project Overview
The Weather Dashboard is a web application that displays real-time weather information for any city entered by the user. It uses asynchronous JavaScript (`fetch` and `async/await`) to retrieve weather data from a public REST API and dynamically updates the webpage.

## ✨ Features
- Search weather by city name
- Displays current temperature
- Displays humidity
- Displays wind speed
- Shows weather description
- Uses Fetch API with `async/await`
- Handles invalid city names and network errors
- Responsive and user-friendly interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- REST API
- Fetch API
- Async/Await

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
4. Enter a city name and click **Search** to view the current weather.

## 📸 Output
The dashboard displays:
- City Name
- Temperature (°C)
- Humidity (%)
- Wind Speed (m/s)
- Weather Description
